这个平台是回测平台

该平台有7大功能模块
Attain_Data:利用爬虫和接口，获取标的的交易数据
Data_Table:主要用于存储csv数据表，避免反复爬取
GUI:设计GUI界面，人性化操作整个框架
Math_Model:数学模型仓库，方便数据预处理和后期建模的需要
Strategy_Functions:完整的策略仓库，方便随时调用和修改
Trade_System:交易系统的代码设计，并且可以自定义回测框架
Visible_Functions:数据可视化仓库，能绘画标的走势图等等

各功能模块的文件介绍：
一、Attain_Data
二、Data_Table
三、GUI
四、Math_Model
五、Strategy_Functions
六、Trade_System
七、Visible_Functions
